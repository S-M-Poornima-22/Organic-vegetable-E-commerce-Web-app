# 🥬 Organic Vegetable E-Commerce Web App

A simple and user-friendly web application for purchasing fresh organic vegetables online. This project provides an easy shopping experience with a clean interface and essential e-commerce features.

## 🌟 Features

* 🛒 Browse organic vegetables
* 🔍 Search for products
* ➕ Add items to cart
* 💳 Simple checkout process
* 📱 Responsive and user-friendly design

## 🛠️ Technologies Used

* Python
* Flask
* HTML5
* CSS3
* JavaScript
